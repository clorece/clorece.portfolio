Minecraft Shader · GLSL — Beta, June 2026

Serie

Path-traced shader pack

Repository Modrinth
Graphics Shader Development Path Tracing Game Modding Volumetric Rendering
GLSL OpenGL 4.3 Iris Shaders API

Built exclusively for Iris Shaders on OpenGL 4.3, Serie brings full path-traced lighting, atmospheric scattering, volumetric cumulus and altocumulus cloud layers, water parallax with vertex displacement, screen-space refraction and reflection, generated normals for glass and ice, Distant Horizons support, integrated PBR, temporal anti-aliasing with upscaling, bloom, automatic exposure, and multiple tone-mapping operators. The repo branch is always the active development version — expect settings and visual behavior to shift between builds.

Until Journey's End

HD-2D Action Roguelike

Repository
Game Development Combat Design System Design AI Implementation Action Roguelike
Unity C# ORCA / RVO2 Unity Input System

Until Journey's End is a fast-paced, zone-based HD-2D action roguelike built in Unity, with aesthetic inspiration drawn from Octopath Traveler. Players fight through randomized arenas, clearing all enemies to spawn the exit portal. A card-based loot system rewards chests with equippable cards — matching suits unlock powerful set bonuses. Enemy AI runs on a custom ORCA/RVO2 integration, enabling intelligent swarming and predictive crowd simulation without clumping. The Saber class features Strife, a high-risk dash-attack skill with stackable kill-streak buffs that scale movement speed and pierce damage. This is an early showcase build demonstrating core mechanics — it does not represent the final game.

Discord Bot + Web App · 2024–2025

Langy

Adaptive language learning platform with AI-powered semantic scoring

Legacy No dedicated repo yet

Langy is an adaptive language learning platform built as both a Discord bot and a standalone web app, designed for casual learners and competitive settings alike. It features AI-powered semantic answer scoring using Sentence Transformers, allowing it to accept contextually correct responses — not just exact matches. A leaderboard system driven by Discord authentication tracked competitive performance across sessions. The backend ran on Hugging Face Spaces; the web frontend was built in React and TypeScript. Full functionality depended on the Spaces instance being live.
